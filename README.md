# pychorewars

A simple task manager with colorized output and JSON storage.

## Installing
\`\`\`bash
pip install colorama
\`\`\`

## Using
# Adding task
python main.py add Buy milk
python main.py add Kill the bill

# View all tasks
python main.py list

# Mark task #1 as completed
python main.py done 1

# Delete task #2
python main.py rm 2

# Clear all
python main.py clear

## Commands
- `add text` — add task
- `list` — view all
- `list --pending` — only active
- `done <id>` — mark ass completed
- `rm <id>` — delete
- `clear` — clear
